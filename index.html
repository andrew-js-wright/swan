swan

Swan is a data-flow driven work-stealing scheduler for multi-core processors. It extends the Cilk/Cilk++/Cilk plus (TM) language by allowing certain inter-dependencies between tasks spawned from the same parent procedure. This way, some tasks are forced to execute serially while others may proceed in parallel. The end result is that the program potentially contains more parallelism and/or is easier to write than an equivalent Cilk program. More information can be found in the following publications:
  • Hans Vandierendonck, George Tzenakis and Dimitrios S. Nikolopoulos. A Unified Scheduler for Recursive and Task Dataflow Parallelism. IEEE Conference on Parallel Architectures and Compilation Techniques (PACT). Accepted for publication. October 2011. [PDF]
  • Hans Vandierendonck, Polyvios Pratikakis, Dimitrios S. Nikolopoulos. Parallel programming of general-purpose programs using task-based programming models. In 3rd USENIX Workshop on Hot Topics in Parallelism (HotPar), Berkeley, California. May 2011. [PDF]
